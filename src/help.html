What this page covers Field definitions, status meanings, and color behavior.

1) What data powers the dashboard

The dashboard is built from the newest capture in Google Drive:

  • NSSC Membership Spreadsheet Capture/SeaScoutShips-YYYY-MM-DD.xlsx

Only the Ships worksheet is used. The publish process:

  1. Keeps rows with a valid UnitID.
  2. Writes data/ships.csv as a snapshot.
  3. Writes data/dashboard-data.json with normalized values.

If you want to check which capture drove a publish, look for: meta.sourceCapture and meta.sourceCaptureName in data/dashboard-data.json.

2) Field sources

Ship identity and basics

  • Ship number → Ship #
  • Ship ID → UnitID (stable dashboard key)
  • Ship name → Ship Name if tracking
  • CST → CST
  • Council → Council Name
  • District → District
  • Owner → Owner (blank shows Unassigned)
  • AUX district → CG Aux Districts (blank shows Unassigned)
  • Focus → Ship Focus (blank shows Unspecified)
  • Ship status → Status (default Established)

Youth, adults, and trends

  • Total youth → Total Youth
  • Adults → Total Adult
  • Primary youth → Primary Youth
  • Total youth YOY → Total Youth YOY
  • Primary youth YOY → Primary YOY

Training and readiness inputs

  • Skipper trained → Skipper Trained
  • Committee Chair trained → CC Trained
  • Size healthy → Size
  • Growth healthy → Growth
  • Advancement healthy → 20% Advance
  • Outdoor healthy → Outdoor
  • Unit Metric → Unit Metric
  • Charter Organization → Charter Organization
  • Skipper Name → Skipper Name

3) How status labels are calculated

  • Size: Healthy when Size is Yes, otherwise Needs Attention.
  • Growth: Healthy when Growth is Yes, otherwise Needs Attention.
  • Advance: Healthy when 20% Advance is Yes, otherwise Needs Attention.
  • Outdoor: Healthy when Outdoor is Yes, otherwise Needs Attention.
  • Training: Complete = both Skipper Trained and CC Trained are Yes; Partial = exactly one is Yes; Gap = neither is Yes.
  • Renewal: Next 120 (within 120 days from refresh), Past (before refresh), Later (after 120-day window), Unscheduled (no valid renewal).
  • Metric: Strong for 4-5, Watch for 2-3, Risk for 0-1.

4) Color meaning

Green = healthy / complete Amber = watch / approaching action Red = needs action Gray = informational (later/unscheduled)

These same colors are used in the health bars, status chips, detail cards, and the board legend.

5) Summary cards and counts

  • Ships: row count with a valid UnitID
  • New ships: rows where Status is New
  • Total youth / adults: sums of Total Youth and Total Adult
  • Average Unit Metric: average of Unit Metric
  • Upcoming renewals: renewal dates in the next 120 days
  • Breakdowns: grouped counts used in charts and filters
